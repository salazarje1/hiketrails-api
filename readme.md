# Hike Trails API

> Backend API for Hike Trails Webpage

## Usage
Rename "config/config.env.env" to "config/config.env" and update the values/setting to your own


## Install Dependencies
```
npm install 
```

## Run App 

```
# Run in dev mode
npm run dev
```

```
# Run in production mode
npm start
```

- Version: 1.0.0
